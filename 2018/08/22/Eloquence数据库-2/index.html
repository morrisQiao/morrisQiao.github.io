1. 安装客户端

下载最新Eloquence B.08.20 版本，安装客户端

https://eloquence.marxmeier.com/download/

2. 数据查询（命令行模式）

2.1. 服务器属性设置

在windows下 打开 cmd ：
set eq_dbserver = 服务器ip地址：端口

例子 ：

set eq_dbserver = 192.168.1.1 : 3390
选择数据库名
b = 数据库名

例子 ：

b = Service
授权部分
PASSWORD => 输入数据库密码

例子 ：

PASSWORD =>  password
选择模式
MODE => 打开模式

例子 ：

MODE => 1 （只读模式）

2.2. 数据查询部分

2.2.1. 简单查询

输入要查询的表名
s = 表名

例子 ：

s = PARAM
构建查询语句
f 表单列名 = 要查询的数值

例子 ：

f idParam = 1235

2.2.2. 模糊查询

f idParam = @
显示结果
r all
[source,perl]

3. 数据导出 （命令行模式）

在windows下 打开 cmd ：
set eq_dbserver = 服务器ip地址：端口

例子 ：

set eq_dbserver = 192.168.1.1 : 3390
