因当前运行环境是 JAVA 8 + TOMCAT 8， 但是javax.ws.rs-api 2.1.1 版本所需的 tag pool 是指向 java 9 的， 在不升级JDK还有Tomcat的情况下，这里只能选择降级， 把版本降到 2.1 版本。

降级到2.1版本后，重新部署重启tomcat，之前异常消失，问题解决了。
