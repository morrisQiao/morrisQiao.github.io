The faintest ink is better than the best memory.
VSCode常用插件
发布于：2019-08-29 22:11:57 标签：/ Work / 访问：

最近从 Atom 改用 VS Code ， 不得不说VS Code真的很强大，比Atom好用太多。 不论是大型文件的加载，还是操作的流顺度，VS Code 都处理的很好。 同时有各种插件可以免费使用，来方便开发。

推荐一些好用的插件 ：

1. Arduino

方便编写查看Arduino程序

2. AsciiDoc

AsciiDoc 文件编写辅助工具， 可以快速生成文档预览。

3. beautify

代码格式化工具， 用起来稍微有点麻烦，但是效果还不错。

4. Code Runner

在VS Code内运行正在编辑的代码程序，支持多种语言。

5. debugger for Chrome

通过设置可实现与chrome联动进行Debug操作， 方便进行前端调试

6. ESlint

可用于js代码的错误检测

7. npm Intellisense

自动补全npm模块名称

8. path Intellisense

智能路径补全工具

9. Project Manager

项目管理工具， 可实现快速在不同项目间进行切换。

10. TypeScript Hero

TypeScript 编程辅助工具

11. vscode-icons

根据每个文件的类型，在每个文件名前加上相应图标。

一目了然，迅速同一个文件夹下不同类型的文件。
